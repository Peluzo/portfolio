# Pasta de Imagens

Coloque sua foto de perfil aqui com o nome `profile.jpg` ou `profile.png`.

Formato recomendado:
- Nome do arquivo: `profile.jpg` ou `profile.png`
- Tamanho ideal: 600x600 pixels (quadrada)
- Formato: JPG ou PNG
- Peso: máximo 500KB (recomendado)

Após adicionar a imagem, atualize o import no arquivo `src/components/Hero/Hero.jsx` se usar um nome diferente ou extensão diferente.
